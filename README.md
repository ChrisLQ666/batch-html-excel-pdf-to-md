# Doc2MD - Document to Markdown Converter

这是一个简单高效的 Python 工具，支持批量将 HTML、Excel、CSV 和 PDF 文件转换为 Markdown 格式。

## ✨ 功能特点
- **HTML**: 自动识别编码（UTF-8/GBK），保留基础格式。
- **Excel**: 支持 `.xlsx` 和 `.xls`，自动处理多个 Sheet。
- **PDF**: 提取文本内容并按页分割。
- **CSV**: 自动转换为 Markdown 表格。
- **GUI**: 启动时弹出文件夹选择框，操作简单。
